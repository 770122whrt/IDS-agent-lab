**IFC4X3_ADD2 Project Specification**

This specification applies to all IfcProject entities in the model.

**Requirements:**

- The model MUST contain exactly one IfcProject instance that serves as the root entity identifying the overall project.

- The IfcProject MUST provide default units for the following measurements:
  - **Length Unit**: Defaults to meters (SI). For imperial systems, inches are recommended.
  - **Angle Unit**: Defaults to degrees. Degrees are the recommended standard unit.
  - **Mass Unit**: Defaults to kilograms (SI). For imperial systems, pounds are recommended.
  - **Temperature Unit**: Defaults to Celsius (SI). For imperial systems, Fahrenheit is recommended.

- The IfcProject MUST maintain a graph of references that connects to all data within the project scope.

- The IfcProject serves as the primary container that establishes project-wide defaults and organizational structure for all model data.
