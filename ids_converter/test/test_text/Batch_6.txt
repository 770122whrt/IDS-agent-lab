【Specification 26】Girder
IFC Version: IFC4X3_ADD2
Description: Bridge girders refer to horizontal support beams that span along the alignment of a bridge.

For steel girders, this refers to each girder line, decomposed into beam segments.
For concrete box girders, this refers to the overall box girder, typically connected to the bridge deck via a keyed construction joint with adjoining reinforcing.

Missing: Alignment, Axis, IfcRelDefinesByType

Applicability:
  Cardinality: 0..unbounded
  • Applies to all IFCELEMENTASSEMBLY type entities

Requirements:
  1. Part Of Requirement
     - Relation Type: IFCRELCONTAINEDINSPATIALSTRUCTURE
     - Related Entity: IFCBRIDGEPART
     - Predefined Type: SUPERSTRUCTURE
     - Cardinality: Required
  2. Attribute Requirement
     - Attribute Name: Name
     - Cardinality: Required
     - Instructions: Name of the girder line for referencing purposes as would be found on construction plans.
  3. Material Requirement
     - Cardinality: Required

【Specification 29】Girder Segment
IFC Version: IFC4X3_ADD2
Description: Girder segments refer to discrete sections along a girder line. They may be modelled as one object that encapsulates the overall cross section (e.g. I-Shape steel, arbitrary precast profile), or as separate objects (e.g. steel flange plates, steel web plate). Each segment may have a constant cross section, a tapered cross section (linearly interpolated from start to end), a variable cross section (linearly interpolated at multiple points between start and end), or free-form geometry.

Missing: Solid Geometry, Connection*, Reinforcing. Tendons, Camber, Stiffeners, Shear Studs

Applicability:
  Cardinality: 1..unbounded
  • Applies to all IFCBEAM type entities

Requirements:
  1. Part Of Requirement
     - Relation Type: IFCRELAGGREGATES
     - Related Entity: Must be one of: IFCELEMENTASSEMBLY, IFCELEMENTASSEMBLYTYPE
     - Predefined Type: GIRDER
     - Cardinality: Required
  2. Attribute Requirement
     - Attribute Name: Name
     - Cardinality: Required
     - Instructions: Name of the girder segment for referencing purposes as would be found on construction plans.
  3. Material Requirement
     - Cardinality: Required
