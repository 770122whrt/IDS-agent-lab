【Specification 16】Material
IFC Version: IFC4X3_ADD2
Description: Materials are defined on elements to be constructed (e.g. concrete), fabricated (e.g. steel girders), and that exist on site (e.g. soil borings), indicating material category, classification, and structural properties.

Applicability:
  Cardinality: 1..unbounded
  • Applies to all IFCMATERIAL type entities

Requirements:
  1. Attribute Requirement
     - Attribute Name: Name
     - Cardinality: Required
     - Instructions: Material classification according to the respective authority (e.g. ASTM)
  2. Attribute Requirement
     - Attribute Name: Category
     - Value: Must be one of: Concrete, Steel, Wood, Plastic, Glass, Earth
     - Cardinality: Required
     - Instructions: Category of material, where if provided must be one of "Steel", "Concrete", "Wood", "Plastic", "Glass", "Earth".
  3. Property Requirement
     - Property Set: Pset_MaterialCommon
     - Property Name: MassDensity
     - Data Type: IFCMASSDENSITYMEASURE
     - Cardinality: Optional
     - Instructions: Material mass density.
  4. Property Requirement
     - Property Set: Pset_MaterialMechanical
     - Property Name: YoungModulus
     - Data Type: IFCMODULUSOFELASTICITYMEASURE
     - Cardinality: Optional
     - Instructions: A measure of the Young's modulus of elasticity of the material.
  5. Property Requirement
     - Property Set: Pset_MaterialMechanical
     - Property Name: ShearModulus
     - Data Type: IFCMODULUSOFELASTICITYMEASURE
     - Cardinality: Optional
     - Instructions: A measure of the shear modulus of elasticity of the material.
  6. Property Requirement
     - Property Set: Pset_MaterialMechanical
     - Property Name: ThermalExpansionCoefficient
     - Data Type: IFCTHERMALEXPANSIONCOEFFICIENTMEASURE
     - Cardinality: Optional
     - Instructions: A measure of the expansion coefficient for warming up the material about one Kelvin.
  7. Property Requirement
     - Property Set: Pset_MaterialConcrete
     - Property Name: CompressiveStrength
     - Data Type: IFCPRESSUREMEASURE
     - Cardinality: Optional
     - Instructions: [If Category=”Concrete”] The compressive strength of the concrete.
  8. Property Requirement
     - Property Set: Pset_MaterialSteel
     - Property Name: YieldStress
     - Data Type: IFCPRESSUREMEASURE
     - Cardinality: Optional
     - Instructions: [If Category=”Steel”] A measure of the yield stress (or characteristic 0.2 percent proof stress) of the material.
