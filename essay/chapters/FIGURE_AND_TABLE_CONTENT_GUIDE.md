## 四、图表创建通用规范

### 4.1 图片规范

| 规范项 | 要求 |
|--------|------|
| 图片尺寸 | 宽度不超过16cm，高度不超过12cm |
| 图片格式 | PNG（推荐）或矢量图（SVG/PDF） |
| 图片编号 | 按章节编号（图3.1、图4.1等） |
| 图片标题 | 置于图片下方，居中，格式为"图X.X 标题" |
| 图片分辨率 | 至少300dpi，确保打印清晰 |
| 图片来源 | 自绘图片无需标注来源；引用图片需标注出处 |

### 4.2 表格规范

| 规范项 | 要求 |
|--------|------|
| 表格格式 | 三线表（顶线粗、栏目线细、底线粗） |
| 表格编号 | 按章节编号（表3.1、表4.1等） |
| 表格标题 | 置于表格上方，居中，格式为"表X.X 标题" |
| 表头样式 | 加粗居中 |
| 内容对齐 | 文字左对齐，数字右对齐或居中 |
| 表格宽度 | 不超过页面宽度，必要时跨页 |

### 4.3 正文引用要求

在正文中引用图表时：
- 图片："如图X.X所示"、"见图X.X"
- 表格："如表X.X所示"、"详见表X.X"
- 确保图表编号与引用一致
